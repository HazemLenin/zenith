````markdown
# Navbar

Top navigation bar.

## Props

| Name  | Type            | Required | Description                       |
| ----- | --------------- | -------- | --------------------------------- |
| title | string          | Yes      | App title/logo                    |
| right | React.ReactNode | No       | Right-side content (e.g., button) |

## Example

```tsx
<Navbar title="Zenith" right={<Button>Login</Button>} />
```
````
