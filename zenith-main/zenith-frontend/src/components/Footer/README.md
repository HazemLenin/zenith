````markdown
# Footer

Bottom section for links or info.

## Props

| Name     | Type            | Required | Description    |
| -------- | --------------- | -------- | -------------- |
| children | React.ReactNode | Yes      | Footer content |

## Example

```tsx
<Footer>
  <div>Links</div>
  <div>Info</div>
</Footer>
```
````
